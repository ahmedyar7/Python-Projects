# Cookie Clicker Automation
This Python script automates the clicking of a cookie object and purchasing upgrades in the cookie-clicking game available at Cookie Clicker. It utilizes the Selenium WebDriver library to interact with the web browser.

## Requirements:-

- Python 3.x
- Selenium WebDriver
- Chrome WebDriver (chromedriver)
- Installation
- Install Python from the official website.
- Install Selenium WebDriver using pip:
```
pip install selenium
```
- Download Chrome WebDriver (chromedriver) from the official website.
- Ensure that the chromedriver executable is in your system's PATH.

## Usage
- Clone or download the script.
- Open the script in a text editor and adjust the timeout values if needed.
- Run the script using Python:
``` 
python cookie_clicker.py
```
